# 🧬 Phidynamics: Framework de Bio-Geometría Fractal

Phidynamics es una librería de investigación que propone una transición de la física lineal a la realidad biológica helicoidal. 

### Principios Fundamentales:
1. **Universo**: Superación del modelo plano mediante curvatura positiva.
2. **Tiempo OAM**: El tiempo no es una línea, es un Momento Angular Orbital.
3. **Torsión Fractal**: La Torsión como campo de información modulado por el número áureo ($\phi$).

Este motor utiliza Geometría de Riemann-Cartan para modelar la autoorganización biológica.
